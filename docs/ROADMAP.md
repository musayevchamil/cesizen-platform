# Roadmap

Target: 21 May 2026.

## Milestone 1 - Setup & Architecture

Scaffold the Laravel + Inertia + Svelte project in `app/`. Set up PostgreSQL locally via Docker Compose, configure Coolify at cesizen.vaycode.com. Sanctum configured for web sessions and API token auth, `/api/v1` route structure in place.

## Milestone 2 - Accounts

Registration, login, and profile management. Role-based access with user and admin roles, plus a basic admin interface for account management.

## Milestone 3 - Information Module

Public content pages visitors can browse without an account. Admins manage content and categories through a CRUD interface.

## Milestone 4 - Emotion Tracker

Users can log, edit, and delete emotion entries, filtered by week or month.

## Milestone 5 - Testing & Delivery

Unit, functional, and regression tests. Validation procedure and acceptance record written up. Technical dossier finalized and demo prepared.
